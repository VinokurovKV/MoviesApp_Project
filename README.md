# Movies App

Web-приложение для просмотра информации о фильмах.

## Функциональность

-   Просмотр списка фильмов
-   Бесконечный скролл
-   Фильтрация:
    -   по рейтингу
    -   по году
    -   по жанрам
-   Страница фильма с подробной информацией
-   Избранное:
    -   добавление через модальное окно
    -   сохранение при перезагрузке
-   Сравнение фильмов:
    -   до 2 фильмов
    -   при выборе третьего --- первый удаляется
    -   таблица сравнения

## Установка и запуск

``` bash
npm install
npm run dev
```

## Переменные окружения

Создать файл `.env` на основе `.env.default`:

    VITE_API_KEY=your_api_key
